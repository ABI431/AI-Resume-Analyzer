# AI Resume Analyzer

An AI-powered Resume Analyzer built using Flask, Google Gemini, HTML, CSS and JavaScript.

## Features

* Resume PDF Upload
* AI Resume Analysis
* ATS Score Evaluation
* Role Fit Analysis
* Job Description Matching
* Skills Detection
* Strengths & Weaknesses Analysis
* Missing Skills Identification
* AI Interview Question Generation
* Modern Responsive Dashboard

## Tech Stack

### Backend

* Python
* Flask
* Google Gemini API
* PDFPlumber

### Frontend

* HTML
* CSS
* JavaScript

## Installation

```bash
git clone <repository-url>
cd AI-Resume-Analyzer

python -m venv venv

venv\\Scripts\\activate

pip install -r backend/requirements.txt
```

Create `.env`

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run:

```bash
python backend/app.py
```

Open:

```text
http://127.0.0.1:5000
```
