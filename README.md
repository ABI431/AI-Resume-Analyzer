# AI Resume Analyzer

## 🚀 Live application 
https://huggingface.co/spaces/abinand-m-3105/ai-resume-analyzer

An AI-powered Resume Analyzer built using:

- Flask
- Google Gemini
- PDFPlumber
- HTML/CSS/JavaScript

## Features

- Resume PDF Upload
- ATS Analysis
- Resume Score
- Job Match Score
- Role Fit Analysis
- Skill Extraction
- Strengths & Weaknesses Detection
- Interview Question Generation

## Tech Stack

### Backend
- Python
- Flask
- Google Gemini API
- PDFPlumber

### Frontend
- HTML
- CSS
- JavaScript

## How It Works

1. Upload a PDF Resume
2. PDF text is extracted using PDFPlumber
3. Google Gemini analyzes the resume
4. The system generates:
   - Resume Score
   - ATS Score
   - Role Fit Score
   - Job Match Score
   - Skills Analysis
   - Strengths & Weaknesses
   - Suggestions
   - Interview Questions
